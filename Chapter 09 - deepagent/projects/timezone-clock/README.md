# Timezone Clock

This project displays the current time in different time zones.

## Setup Instructions

```bash
python -m venv venv
# Linux/Mac call
source venv/bin/activate
# Windows call
venv\Scripts\activate
pip install -r requirements.txt
```

## How to Run the Program

Run the application using:

```bash
python main.py
```

## Example Output

```
Current time in America/New_York: 2023-10-08 15:20:35
Current time in Europe/London: 2023-10-08 20:20:35
Current time in Asia/Tokyo: 2023-10-09 05:20:35
```
